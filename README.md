## 🎒 外出前持ち物チェッカー「Bag Check v1」
本プロトタイプは `/bag-check-v1/index.html` から動作を確認できます。

### 1. 何のバグを解決するプロダクトか
- 朝、家を出る直前に財布やイヤホンなどの持ち物を探して出発が遅れたり、必要な物を忘れて外出先で困るというバグ（課題）。

### 2. 誰のためのプロダクトか
- 外出前に忘れ物をしやすい人、出発前にバタバタしがちな人（例：通学する学生やバイトをする人）。

### 3. v1で何が動くか
- **場面選択**: 「学校」「バイト」「遊び」「雨の日」の4つの場面をワンタップで切り替え。
- **場面別リスト表示**: 場面に応じた最適なデフォルト持ち物リストを表示。
- **チェックボックス**: 各アイテムをタップでチェック（チェック時は取り消し線のアニメーション）。
- **進捗のリアルタイム表示**: 「残り何個か」をカウントし、プログレスバーを動的に伸縮。
- **準備完了演出**: 全てのチェックが完了した際、お祝いのモーダルメッセージを表示。
- **マイアイテム追加・削除**: リストにない独自の持ち物をその場で追加、または不要なアイテムを削除可能（データはブラウザの `localStorage` に保存され永続化）。

### 4. 今後追加したい機能 (ロードマップ)
- **カスタムテンプレート作成**: ユーザーが自分で場面（例：「旅行」「ジム」など）を新規作成できる機能。
- **チェック時間のリマインダー**: 出発予定時刻の10分前に自動でチェックを促すプッシュ通知機能。
- **スマート提案 (AIアシスト)**: 天気予報APIやカレンダー連携から、その日に必要な持ち物（傘、ハンカチ、特定の教科書など）をAIが自動提案する機能。

## 受講生用テンプレートの使い方

### 第3回(初回・Use this template)

1. このリポジトリの **緑の "Use this template" ボタン** で自分のリポジトリを作成

2. 自分のリポジトリの [`lectures/lecture3/README.md`](./lectures/lecture3/) を開く
3. そこに常設の **Antigravity 用固定プロンプト** をコピペ → 内容を埋めて Antigravity で実行
4. 自動で `lectures/lecture3/` の3ファイル更新 + commit + push
5. ポータルアプリ(第3回 課題提出欄)に GitHub レポURL を提出

### 第4回以降(累積運用)

1. **同じリポジトリ** を Antigravity で開く(Use this template は不要)
2. このリポジトリの `lectures/lecture[N]/README.md` から **第N回プロンプト** をコピペ
3. プロンプトが pub-docs から第N回フォルダを fetch → 自リポに `lectures/lecture[N]/` 追加 → 内容反映 → push
4. 同じレポURLをポータルで提出

→ **学生は「README からプロンプトをコピペ」だけ**(毎回同じパターン・新規レポ作成不要)
